## 1. Conbine.py

### 🎬 동영상 파일 합치기 & 무작위 셔플 (Python + FFmpeg)

💡 이 스크립트는 무엇인가요?
이 파이썬 스크립트는 지정된 폴더에 있는 여러 개의 동영상 파일을 하나의 파일로 빠르게 합쳐주는 도구입니다. 특히, 합치는 순서를 무작위로 섞을(셔플) 수 있는 기능이 포함되어 있습니다.

주요 기능:
특정 폴더(videos) 내의 모든 동영상 파일(.mp4, .mov, .avi, .mkv 등)을 찾습니다.
파일 목록을 무작위로 섞거나 (기본 설정), 순서대로 정렬할 수 있습니다.
FFmpeg 도구를 사용하여 이 동영상들을 하나로 합칩니다.
인코딩 없이 원본 파일을 복사(-c copy)하는 방식으로 처리 속도가 매우 빠릅니다.

### 🛠️ 준비 사항 (필수)
이 스크립트는 강력한 미디어 처리 도구인 FFmpeg을 사용합니다. 스크립트를 실행하기 전에 반드시 다음 두 가지를 준비해야 합니다.

1. Python 설치
스크립트 실행을 위해 Python 3가 설치되어 있어야 합니다.

2. FFmpeg 설치 및 경로 설정
FFmpeg을 다운로드하여 설치하고, 시스템 환경 변수(PATH)에 ffmpeg 명령어가 인식될 수 있도록 경로를 설정해야 합니다.

설치 확인: 터미널/명령 프롬프트에서 ffmpeg -version을 입력했을 때 버전 정보가 출력되면 성공입니다.

### 🚀 사용 방법 및 작동 순서

1. 프로젝트 구조 만들기
스크립트 파일(combine.py 등)이 있는 동일한 위치에 합칠 동영상들을 담을 폴더를 만듭니다. (예, videos 폴더)
🎬 동영상 병합 프로젝트 구조 및 설정


### 🎥 동영상 파일 준비

- 병합하고 싶은 모든 동영상 파일을 `videos` 폴더 안에 넣습니다.
- 다양한 확장자(mp4, mov, mkv 등)를 지원합니다.

### ⚙️ 설정 변수

| 변수 이름         | 설명                                 | 기본값                              |
|------------------|--------------------------------------|-------------------------------------|
| `video_folder`   | 동영상 파일이 들어있는 폴더 이름     | `'videos'`                          |
| `output_filename`| 최종 합쳐진 동영상 파일 이름         | `'merged_video_ffmpeg_shuffled.mp4'`|
| `shuffle_videos` | 동영상 순서를 무작위로 섞을지 여부   | `True`                              |

### 🛠️ 사용 방법

1. `videos` 폴더에 병합할 동영상 파일들을 넣습니다.
2. `combine.py` 파일을 실행합니다.
3. 설정한 옵션에 따라 동영상이 병합되어 출력됩니다.

### 💡 참고 사항

- FFmpeg가 시스템에 설치되어 있어야 합니다.
- 파일 이름에 공백이나 특수문자가 있을 경우 오류가 발생할 수 있으므로 주의하세요.


### 💡 셔플 끄는 방법

`shuffle_videos = False`로 설정하면 `videos` 폴더에 있는 파일 목록 순서대로 병합됩니다.

### 🖥️ 스크립트 실행

터미널 또는 명령 프롬프트를 열고, `combine.py` 파일이 있는 프로젝트 폴더로 이동한 후 다음 명령어를 실행합니다:

```bash
python merge_videos.py
```

✅ 결과 확인
스크립트 실행이 완료되면, 프로젝트 폴더 안에 설정한 이름(merged_video_ffmpeg_shuffled.mp4)으로 하나로 병합된 동영상 파일이 생성됩니다.
